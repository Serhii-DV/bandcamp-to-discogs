About B2D

Bandcamp to Discogs (B2D) is a Google Chrome extension designed to assist in converting a Bandcamp artist release page into a draft CSV file for Discogs.

How it works

The Discogs website has a feature called CSV_To_Draft that allows you to create drafts from a CSV file.

The main idea of this extension is to extract release data from the Bandcamp page and prepare a CSV file that can be imported into Discogs.

How to use

  1. Visit any Bandcamp artist release page.
  2. Click on the Bandcamp to Discogs extension. If there are no errors, you will see release information in the format: Artist / Album / Year / Tracklist.
  3. You can preview the generated Discogs CSV data by clicking on the second button: (Preview Discogs CSV data).
  4. Click on the third button: (Download Discogs CSV file), and save the locally generated CSV file.
  5. Click on the fourth button: (Open Discogs CSV to Draft page and start import). It will open the Discogs CSV_To_Draft page, where you must choose and import the saved CSV file.
  6. Go to the Drafts page and edit your imported release.
