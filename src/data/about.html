About Bandcamp to Discogs Extension

Bandcamp to Discogs (B2D) is a Google Chrome extension designed to assist in converting a Bandcamp artist release page into a draft CSV file for Discogs.

How it works

The Discogs website has a beta feature called CSV_To_Draft that allows you to create drafts from a CSV file.

The main idea of this extension is to extract release data from the Bandcamp page and prepare a CSV file that can be imported into Discogs.

How to Use Bandcamp to Discogs Extension

  1. Go to a Bandcamp artist's release page.
  2. Click on the Bandcamp to Discogs extension.
  3. If everything is okay, you'll see information about the release in this order:
    Artist / Album / Year / Tracklist
  4. You can see a preview of the information by clicking the second button (Preview Discogs CSV data)
  5. To save the information, click the third button (Download Discogs CSV file). Keep this file on your computer.
  6. Click the fourth button (Open Discogs CSV to Draft page and start import). This will take you to the Discogs CSV_To_Draft page where you choose and bring in the saved CSV file.
  7. Go to the Drafts page and click to edit the release you imported. Most of the important information will be filled in automatically by the B2D extension, but remember to double-check if it's correct. You can also add more details like release credits, tracks credits, etc.
